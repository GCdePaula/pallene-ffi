# A Foreign Function Interface For Pallene

This repository contains my work on an implement of a foreign function interface for Pallene:

* **master's thesis** — published on September 2021, and can be found [here](http://www.lua.inf.puc-rio.br/publications/2021-GabrielDePaula-msc-dissertation.pdf);
* **peer-reviewed paper** — published in _SBLP '22: Proceedings of the XXVI Brazilian Symposium on Programming Languages_, and can be found [here](https://dl.acm.org/doi/10.1145/3561320.3561321);
* **implementation** — added as a git submodule in this repository.
